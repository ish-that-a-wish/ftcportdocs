# Lesson Audio Files

Place recorded audio files here to replace the browser text-to-speech fallback.

## Naming Convention

```
lesson-{number}.mp3
```

| File              | Lesson                          |
| ----------------- | ------------------------------- |
| `lesson-0.mp3`   | Lesson 1: How to Use This Tutorial  |
| `lesson-1.mp3`   | Lesson 2: General Requirements      |
| `lesson-2.mp3`   | Lesson 3: Best Practices            |
| `lesson-3.mp3`   | Lesson 5: Design Process            |
| `lesson-4.mp3`   | Lesson 4: Team Overview             |
| `lesson-5.mp3`   | Lesson 6: CAD & Technical Drawings  |
| `lesson-6.mp3`   | Lesson 7: Robot Subsystems          |
| `lesson-7.mp3`   | Lesson 8: Software & Programming    |
| `lesson-8.mp3`   | Lesson 9: Business & Community      |
| `lesson-9.mp3`   | Lesson 10: Results & Season Summary |

## Supported Formats

- **`.mp3`** (recommended: widest browser support)
- `.ogg` and `.wav` also work but `.mp3` is preferred

## Recording Tips

- Read the full lesson text naturally at a comfortable pace
- Keep a consistent volume and tone across lessons
- Aim for clear pronunciation of technical terms
- No background music needed: keep it clean
